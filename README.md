# Traffic Violation Detection & Notification System

## Overview

This project is an AI-based traffic monitoring system designed to automatically detect traffic violations using computer vision techniques.

The system processes video input, detects vehicles, identifies violations such as riding without a helmet and wrong-side driving, extracts vehicle numbers using OCR, and sends SMS notifications to vehicle owners.

It also maintains logs of all violations for analysis and reporting.

This project demonstrates a real-world application of Artificial Intelligence in smart traffic management.

---

## Features

-  Detects vehicles using YOLOv8
-  Identifies helmet violations for motorcyclists
-  Detects wrong-side driving using movement tracking
-  Extracts license plate numbers using OCR (EasyOCR)
-  Sends SMS alerts using Twilio API
-  Logs violations in CSV and Excel format
-  Saves images of detected violations
-  Generates output video with highlighted detections
